# prevencion-y-acompa-amiento-digital-para-tutores
Plataforma Web de Prevención y Acompañamiento Digital, una iniciativa diseñada con un propósito claro: empoderar a padres, tutores y adultos responsables con herramientas accesibles, información confiable y protocolos de acción inmediata para proteger a los menores en internet.
